# Mapeo: Web pública ↔ Dashboard administrador

**Verificación:** Todo el contenido visible en la web puede editarse desde el panel de administración.

| Qué se ve en la web | Dónde se edita en el admin | Ruta admin |
|---------------------|----------------------------|------------|
| Hero (carrusel inicio: imágenes + textos) | Imágenes Hero + Configuración (pestaña Hero) | `/admin/imagenes`, `/admin/configuracion` |
| Sección Servicios (inicio) | Servicios | `/admin/servicios` |
| Página /servicios y /servicios/[slug] | Servicios | `/admin/servicios` |
| Sección Proyectos / portfolio (inicio) | Proyectos | `/admin/proyectos` |
| Página /proyectos | Proyectos | `/admin/proyectos` |
| Sección "Por qué CRONEC" y "Proceso" (inicio) | Secciones de Inicio | `/admin/secciones` |
| Testimonios (inicio) | Testimonios | `/admin/testimonios` |
| Certificaciones y Clientes (inicio) | Certificaciones y Clientes | `/admin/certificaciones-clientes` |
| Blog / Noticias | Noticias | `/admin/noticias` |
| Página /nosotros | Nosotros + Configuración (misión/visión) | `/admin/nosotros`, `/admin/configuracion` |
| Datos empresa (footer, contacto, layout) | Configuración | `/admin/configuracion` |
| Formulario de contacto → mensajes | Mensajes | `/admin/mensajes` |
| Calculadora / cotizador | Cotizador | `/admin/cotizador` |

**Conclusión:** Toda la web es administrable desde el dashboard.
